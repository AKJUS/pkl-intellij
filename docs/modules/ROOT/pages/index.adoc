= IntelliJ Plugin

The IntelliJ plugin provides Pkl language support for the IntelliJ Platform (IntelliJ IDEA, GoLand, PyCharm, etc.).
It requires [[min-intellij-version]]**IntelliJ 2023.1** or higher and supports **Pkl 0.25** and higher.
